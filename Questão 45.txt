45. Como eu poderia resolver conflitos ocorridos utilizando-se oworkflow básico do git?
basicamente deverá ser feito o seguinte processo:
Passo 1: Baixar novamente o repositório em um novo diretório
Passo 2: Copiar todos os arquivos, do diretório antigo, exceto o arquivo com conflito;
Passo 3: Editar o arquivo com conflito no novo diretório
Passo 4: git add –u ; git commit –m “sua mensagem”; git push.
